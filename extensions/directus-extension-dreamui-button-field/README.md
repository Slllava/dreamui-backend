# DreamUI Button Field

Directus interface extension for editing button configs in a single string field.

Stored value example:

```json
{
  "title": "Read more",
  "target": "_self",
  "urlType": "custom_url",
  "page": "",
  "customUrl": "https://example.com",
  "otherUrl": "home",
  "style": "primary",
  "icon": "{\"library\":\"lucide\",\"name\":\"ArrowRight\",\"strokeWidth\":2}"
}
```

If `dreamui-icon-field` is installed, the icon picker is embedded automatically.
